伪元素

发表于2025-02-18|更新于2025-02-21|CSS
|阅读量:

::before 和 ::after 是 CSS 中的伪元素选择器，它们允许你在元素的内容前或后插入虚拟的“元素”（不会出现在 DOM 中），用于装饰或添加额外内容。以下是详细解析：

一、基本语法

二、核心特性

特性 说明
虚拟元素 不会出现在 HTML 结构中，仅通过 CSS 渲染。
依赖 content 必须设置 content 属性（即使为空 content: "";），否则不生效。
可样式化 可以添加文字、图标、背景、动画等任何样式。
默认行内元素 默认是 display: inline，可通过 CSS 改为块级或定位。

三、使用场景

1. 添加装饰性内容

2. 视觉增强（如悬浮效果）

3. 清除浮动（经典用法）

4. 纯 CSS 图标或形状

四、注意事项

  1. content 的值：

    • 文本：content: "文字";（需用引号包裹）。

    • 空内容：content: "";（常用于纯装饰）。

    • 属性值：content: attr(data-tip);（显示元素的 data-tip 属性值）。

    • 图标：content: url(icon.png);。

  2. 定位控制：

    • 若需要绝对定位，需将父元素设为 position: relative。
.parent {
  position: relative;
}
.parent::before {
  content: "";
  position: absolute;
  top: 0;
  left: 0;
}
  1. 性能优化：
    • 避免在大量元素上使用复杂伪元素动画，可能导致渲染性能问题。

五、与单冒号 :before 的区别

  • 语法规范：CSS3 规定伪元素用 ::（如 ::before），伪类用 :（如 :hover）。
  • 兼容性：大部分浏览器支持单冒号写法，但推荐使用 :: 以符合标准。
